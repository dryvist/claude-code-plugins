# agent-teams-orchestrator

Claude Code plugin for managing Agent Team workflows with reusable team templates.

## Skills

- **`team-templates`** — Pre-configured team templates for common Agent Team workflows

## Templates

- **review-team** — Multi-perspective code review (security, performance, test, quality lenses)
- **debug-team** — Competing hypothesis debugging
- **feature-team** — Parallel feature development with architect gate
- **repo-ops-team** — Multi-repository bulk operations
- **refactor-team** — Large-scale refactoring with file-ownership isolation

See `skills/team-templates/SKILL.md` for full specifications, model routing, task patterns, and sizing guidance.
