# homelab-ops

High-level operational runbooks for homelab management: DR-node power
management, DNS ingress convergence, and secrets-engine identity bring-up.

## Skills

- **`/homelab-runbooks`** - Power-managed DR node wake (Dell PowerEdge/iDRAC), DNS ingress-alias record convergence, and OpenBao/Terrakube identity bring-up order

## Installation

```bash
claude plugins add jacobpevans-cc-plugins/homelab-ops
```

## Usage

```text
/homelab-runbooks
```
